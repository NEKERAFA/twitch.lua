twitch:connectIRC ([nickname[, token]])
Connect to IRC server. If nickname and token is omited, it use an only-read connection.

Parameters:

  • nickname the Twitch username in IRC server. (optional)
  • token the access token to authenticate with IRC server. (optional)

Send a IRC message.

twitch:receive (channel)
Get next IRC message received.

Parameters:

  • channel [opt] the channel to filter.

Returns:

    nil if there is any message receive, otherwise return a message object with following values:
    • date The date when the message is received as ISO 8601 string.
    • tags Twitch IRC tags if you request the irc capabilities.
    • nickname The nickname of the person who send the message.
    • channel The channel which message was sended.
    • message The message as string.
